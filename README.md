Descrizione del progetto:

Il progetto consiste nella ricreazione dell’interfaccia di Telegram usando solo HTML e CSS, senza funzionalità reali di invio messaggi.

Abbiamo creato una lista di chat che simula i contatti e i gruppi.

È possibile cliccare su un contatto o gruppo per aprire la relativa chat, anche se la chatbar è simulata e non permette di scrivere.

Lo stile grafico dell’interfaccia, comprese le chat e la schermata principale, è stato realizzato interamente con CSS.

L’obiettivo del progetto è ricreare visivamente l’esperienza di Telegram, mostrando l’organizzazione di chat e contatti in un’app di messaggistica moderna.
